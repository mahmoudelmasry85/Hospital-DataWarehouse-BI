# Hospital Data Warehouse & Business Intelligence Project

## Project Description
This project demonstrates the design and implementation of a complete Data Warehouse and Business Intelligence solution
for a Hospital Management System.

## Business Problem
Hospitals generate large amounts of operational data related to patients, doctors, visits, and treatments.
This project helps management analyze hospital performance, costs, and doctor efficiency using a centralized Data Warehouse.

## Data Sources
- CSV files (Patients, Doctors, Departments, Visits)
- Excel files (if applicable)

## Data Warehouse Design
- Star Schema Architecture
- Fact Table: FactVisits
- Dimension Tables:
  - DimPatient
  - DimDoctor
  - DimDate
  - DimDepartment

## ETL Process
- Data extraction from CSV/Excel
- Data transformation using SSIS
- Loading data into staging and data warehouse schemas

## BI & Analytics
- Interactive Power BI dashboards
- KPIs: Total Visits, Total Revenue, Average Treatment Cost
- Analysis by Doctor, Department, and Time

## Tools & Technologies
- Microsoft SQL Server
- SSIS
- Power BI
- GitHub
